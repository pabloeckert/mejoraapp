## Sprint 1 — UX Crítico (05:27-05:32)

### 1.1 Targets táctiles header
- **Archivo:** `AppHeader.tsx`
- **Antes:** h-8 w-8 (32×32px) — debajo del mínimo de 44×44px
- **Ahora:** h-11 w-11 (44×44px) + iconos w-5 h-5

### 1.2 Errores humanos en login
- **Archivo:** `LoginForm.tsx`
- **Función:** `humanizeAuthError()` — mapea errores de Supabase a mensajes claros
- **Mapeos:** invalid credentials, email not confirmed, too many requests, user not found, network

### 1.3 Errores humanos en registro
- **Archivo:** `SignupForm.tsx`
- **Mapeos:** already registered, password short, invalid email, too many requests

### 1.4 Secuencia de modales
- **Archivo:** `Index.tsx`
- **Antes:** Onboarding y ProfileCompleteModal podían mostrarse juntos
- **Ahora:** Onboarding solo si profileComplete === true (primero profile, luego onboarding)

### 1.5 Loading states descriptivos
- **Archivo:** `Index.tsx`
- **Antes:** Mismo spinner para "cargando sesión" y "verificando perfil"
- **Ahora:** Texto descriptivo debajo del spinner en cada caso

### 1.6 Tab label más corto
- **Archivo:** `BottomNav.tsx`
- **Antes:** "Contenido" (largo, puede truncarse)
- **Ahora:** "Tips" (corto, claro)

### 1.7 Dark mode progress bar
- **Archivo:** `DiagnosticTest.tsx`
- **Antes:** `bg-white/25` (blanco en dark mode)
- **Ahora:** `bg-foreground/15` (theme-aware)

### 1.8 Pull-to-refresh
- **Archivo:** `Muro.tsx`
- **Implementación:** Touch events (start/move/end) + threshold de 80px
- **Feedback visual:** ArrowDown rotacional + texto "Jalá/Soltá/Actualizando"

### 1.9 Textarea comentarios
- **Archivo:** `Muro.tsx`
- **Antes:** 36px de altura
- **Ahora:** 44px (target táctil más usable)
